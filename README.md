## ✨ Features

- **Google Authentication:** Secure and persistent login using NextAuth.js v5 and Google OAuth.
- **Digital Wardrobe:** Visually store all your clothing items in a fully mobile-optimized smart grid interface.
- **Smart Image Uploads:** Live Cloudinary integration supports clicking a picture from a mobile camera or uploading an existing file directly into your wardrobe.
- **Laundry Tracking:** Select one or multiple items, send them to the "laundry status", and set distinct expected return dates.
- **Beautiful Glassmorphism UI:** Built from the ground-up strictly utilizing Tailwind CSS, modern dark mode design tokens, smooth Framer Motion animations, and touch-friendly interactions (e.g. Bottom Navigation layout).
- **Progressive Web App (PWA):** `manifest.json` included, allowing the app to be seamlessly pinned and installed like a native mobile app on iOS/Android.

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router, Server API Routes)
- **Database:** MongoDB & Mongoose
- **Authentication:** Auth.js / NextAuth.js v5 (Edge Middleware Compatible)
- **Styling:** Tailwind CSS & Framer Motion
- **Storage:** Cloudinary SDK
- **Language:** TypeScript 5
